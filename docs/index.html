Design Decisions

ADR-001
Go for WebSocket, Python for ML
Goroutines handle concurrent connections with minimal overhead. Python has no viable alternative for MediaPipe, faster-whisper, and torch-based ML libraries.
ADR-002
gRPC replacing stdout IPC for vision
Typed proto contract; bidirectional streaming; priority interrupt support. Stdout IPC had no schema enforcement and no cancellation primitive.
ADR-003
webrtcvad over Silero VAD
No torch dependency; native ARM64; processes 30ms chunks in under 1ms. Silero caused dependency conflicts on Apple Silicon.
ADR-004
Native ARM64 Python via miniconda
Avoids Rosetta x86_64 emulation conflicts. Unlocks Apple Neural Engine acceleration for all ML inference.
ADR-005
MediaPipe Tasks API
Solutions API deprecated in 0.10.21+. Tasks API provides Metal GPU acceleration and cleaner interface.
ADR-006
NATS for async transport
Decoupled event bus eliminates Go channel fan-out complexity. Subjects map cleanly to vision/audio/cognition streams.
ADR-007
Claude Haiku / Sonnet tiered routing
Cost-sensitive fast path (Haiku) with Sonnet escalation for complex turns. Prompt caching on stable system prefix reduces latency further.
ADR-008
NetworkX graph memory alongside ChromaDB
Relational traversal for memory anchoring is not achievable via vector similarity alone. Graph edges encode explicit semantic relationships.

cd aria
cp backend/.env.example backend/.env
# Add your ANTHROPIC_API_KEY to backend/.env

2. Python Environment (Apple Silicon)

bash 
curl -O https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-arm64.sh
bash Miniconda3-latest-MacOSX-arm64.sh -b -p ~/miniconda-arm64
source ~/miniconda-arm64/bin/activate

pip install mediapipe opencv-python torch faster-whisper webrtcvad \
            sounddevice structlog pydantic pydantic-settings \
            anthropic fastapi uvicorn chromadb deepfilternet networkx
Always use /Users/YOUR_USERNAME/miniconda-arm64/bin/python3 — never system Python or conda base.

3. Backend & Frontend

bash 
cd backend && go mod download
cd ../frontend && npm install

4. Run (three terminals)

Terminal 1 — FastAPI cognition + memory service 
cd backend
PYTHONPATH=backend /Users/YOUR_USERNAME/miniconda-arm64/bin/python3 \
  -m uvicorn app.main:app --port 8000
Terminal 2 — Go server (gRPC vision worker + NATS) 
cd backend
go run cmd/server/main.go
Terminal 3 — Frontend 
cd frontend
npm run dev

Open http://localhost:3000 and allow camera and microphone access.

Ongoing Constraints

127.0.0.1 only — never 0.0.0.0 for gRPC backend/.env is gitignored — never commit API keys never hardcoded Security findings are P0 — fix before any sprint merge No security debt carried forward between versions

v3 Security Requirements

  • mTLS on gRPC services when ARIA exposes to local network
  • WebSocket rate limiting: max N connections per IP
  • Session token expiry: UUID sessions expire after 30 min idle
  • Audit log: all cognition requests logged with session_id + timestamp
  • Input validation: sanitize all WebSocket message payloads before routing

v4 Security Requirements (Agent Trust Boundaries)

  • Segregation of duties enforced via gitagent DUTIES.md
  • code-builder agent cannot approve its own work
  • Agent-to-agent communication authenticated (not just localhost trust)
  • API key rotation: ANTHROPIC_API_KEY rotated every 90 days
  • No agent can write to SOUL.md without human approval (human-in-the-loop)
